## 📚 项目简介

Reader 是一款为热爱阅读的用户精心打造的现代化电子书阅读器，致力于提供纯粹、智能的阅读体验。通过集成先进的 AI 能力和支持丰富的电子书格式，让阅读变得更加智能、专注和高效。

![](./docs/images/main.jpg)

## ✨ 核心功能

| 功能模块 | 详细说明 | 状态 |
| --- | --- | --- |
| **多种格式支持** | 全面支持 EPUB、MOBI、AZW3、FB2、TXT、PDF 等主流电子书格式 | ✅ |
| **全平台数据同步** | 覆盖 Android、iOS、macOS、Windows 多端，实现书籍文件、笔记、阅读进度的无缝同步 | ✅ |
| **智能 AI 助理** | 按阅读进度与风格智能整理书架，生成思维导图辅助理解，提供 AI 词典与即时翻译，以及内容观点分析与总结 | ✅ |
| **个性化阅读体验** | 自定义字间距、段间距、行间距与边距，调整字体大小、样式与字重，配置阅读配色与背景图片，设置对齐方式与自定义样式 | ✅ |
| **丰富的笔记功能** | 支持多配色与样式选择，按时间、章节排序并可按颜色筛选，导出 TXT、Markdown、CSV 等多种格式，一键生成美观卡片便于分享 | ✅ |
| **详细的阅读统计** | 记录阅读时长，按年/月/周/日多维度查看，通过阅读热力图直观呈现阅读习惯变化 | ✅ |
| **实用附加功能** | 听书功能（支持多模型、语速、音色与定时），书籍全文翻译（原文、译文对照阅读），云端存储节省本地空间，中文简繁体一键转换 | ✅ |
| **OPDS 书源** | 支持 OPDS 书源，可自定义添加 | 🛠️ |

## 🔧 从源码构建

希望从源码构建 Anx Reader？请按照以下步骤操作：

1. 安装 [Flutter](https://flutter.dev) 开发环境
2. 克隆项目仓库并进入项目目录
3. 运行 `flutter pub get` 安装依赖
4. 运行 `flutter gen-l10n` 生成多语言文件
5. 运行 `dart run build_runner build --delete-conflicting-outputs` 生成 Riverpod 代码
6. 运行 `flutter run` 启动应用

> **注意**：您可能会遇到 Flutter 版本兼容性问题，请参考 [Flutter 官方文档](https://flutter.dev/docs/get-started/install) 进行解决。
